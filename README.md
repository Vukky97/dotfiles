Please, check the [wiki](https://github.com/t0thkr1s/dotfiles/wiki/Installation) page!

 - Font : [SF Mono](https://github.com/ZulwiyozaPutra/SF-Mono-Font)

 - Activity Monitor : [gotop](https://github.com/cjbassi/gotop)
